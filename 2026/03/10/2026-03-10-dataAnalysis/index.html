# 一、截位直除

# 方法概述

“截位直除”是资料分析中最常用的速算方法之一，核心目的是：通过保留前两位或前三位有效数字，快速比较或估算结果。

# 使用规则

  • 截谁：

    • 一步除法截分母
    • 多步除法上下都截

  • 选项差距：

    • 差距大，截两位：
      • 首位均不同
      • 首位相同时，次位差大于首位
    • 差距小，截三位：
      • 首位相同，次位差小于等于首位

# 补充说明

  • 选项数字不同时，截位不考虑量级
  • 选项数字相同时（如 0.88%、8.8%），要判断量级
  • 若选项前三位数字都一样，此时至少需要截 4 位
  • 这种情况一般属于精算，较少出现

# 方法示例

# 1. 一步除法截分母

求：

678223\frac{678}{223} 223678​

可直接截分母：

678223≈678220≈3.08\frac{678}{223} \approx \frac{678}{220} \approx 3.08 223678​≈220678​≈3.08

若选项为 2.1、3.1、4.2、5.3，可直接选 3.1。

# 2. 多步除法上下都截

求：

67892345\frac{6789}{2345} 23456789​

上下都截：

67892345≈6823≈2.96\frac{6789}{2345} \approx \frac{68}{23} \approx 2.96 23456789​≈2368​≈2.96

若选项差距较大，足够选出答案。

# 3. 差距大，截两位

求增长率：

135543\frac{135}{543} 543135​

选项为 12%、24%、36%、48%，差距很大，可截两位：

135543≈1354≈24%\frac{135}{543} \approx \frac{13}{54} \approx 24\% 543135​≈5413​≈24%

直接选 24%。

# 4. 差距小，截三位

求：

284731\frac{284}{731} 731284​

选项为 38.1%、38.6%、39.1%、39.6%，差距较小，需要截三位：

284731≈38.8%\frac{284}{731} \approx 38.8\% 731284​≈38.8%

更接近 39.1%。

# 5. 量级要判断

若算得：

8810000=0.0088=0.88%\frac{88}{10000}=0.0088=0.88\% 1000088​=0.0088=0.88%

若选项中同时出现 0.88% 和 8.8%，此时不能只看数字 88，必须看小数点位置，答案应为 0.88%。

# 6. 前三位一样，至少截 4 位

例如四个选项为：

  • 12.31%
  • 12.34%
  • 12.37%
  • 12.39%

若计算式为：

152712384\frac{1527}{12384} 123841527​

前三位都可能落在 12.3 附近，此时至少要再多保留一位，不能只截两位三位，否则区分不开。

# 易错提醒

截位直除的前提是：只需要和选项比大小，不必追求绝对精确值。
做题时优先服务于“选项区分”，不要陷入死算。

# 二、分数比较

# 方法概述

分数比较本质上就是比较“每一份有多少”。

# 核心原则

  • 分子大且分母小，分数大
  • 即：钱多人少，结果更大

# 特殊情况

  • 分子分母同大同小时：
    • 竖着直接除

# 技巧总结

  • 横着看倍数，谁的倍数大就看谁
  • 分子大的分数大，分母大的分数小
  • 若横着看不出来，就竖着简单除一下

# 方法示例

# 1. 分子大且分母小，分数更大

比较：

45120和42130\frac{45}{120} \quad \text{和} \quad \frac{42}{130} 12045​和13042​

前者分子更大、分母更小，因此：

45120>42130\frac{45}{120}>\frac{42}{130} 12045​>13042​

# 2. 分子分母同大同小时，竖着除

比较：

38125和41140\frac{38}{125} \quad \text{和} \quad \frac{41}{140} 12538​和14041​

一个分子大、分母也大，看不出时可直接除：

38125=0.304\frac{38}{125}=0.304 12538​=0.304

41140≈0.293\frac{41}{140}\approx0.293 14041​≈0.293

所以：

38125>41140\frac{38}{125}>\frac{41}{140} 12538​>14041​

# 3. 横着看倍数

比较：

2480和30120\frac{24}{80} \quad \text{和} \quad \frac{30}{120} 8024​和12030​

分子从 24 到 30，扩大了：

3024=1.25\frac{30}{24}=1.25 2430​=1.25

分母从 80 到 120，扩大了：

12080=1.5\frac{120}{80}=1.5 80120​=1.5

分母扩大更快，所以分数变小：

2480>30120\frac{24}{80}>\frac{30}{120} 8024​>12030​

# 易错提醒

“分子分母都变大”或“都变小”时，不能只看一个数。
看不清时，直接除，比硬猜更稳。

# 三、量相关

# 1. 基期量（必考）

# 识别

  • 求过去的量
  • 求去年、上年、前期的量

# 公式

  1. 基期量=现期量−增长量基期量 = 现期量 - 增长量基期量=现期量−增长量

  1. 加减法精确计算，可用尾数

  3. 基期比较：现期大且 rrr 小的，基期更大

# 方法示例

某市今年工业增加值为 540 亿元，同比增长 8%，求去年工业增加值。

684−127=557684-127=557 684−127=557

直接得去年产量为 557 万吨。
若选项尾数分别为 553、557、562、568，甚至可以先看尾数快速锁定 557。

# 3. ∣r∣≤5%|r|\le5\%∣r∣≤5%，化除为乘

某地区今年零售额为 210 亿元，同比增长 4%，求去年零售额。

严格算是：

2101.04\frac{210}{1.04} 1.04210​

因为 4%4\%4% 很小，可近似看成：

210×(1−4%)=210×0.96=201.6210\times(1-4\%)=210\times0.96=201.6 210×(1−4%)=210×0.96=201.6

210÷1.04≈201.9210\div1.04\approx201.9 210÷1.04≈201.9

近似非常接近，做选择题够用。

# 4. 基期比较：现期大且 rrr 小，基期更大

甲：现期 220，同比增长 10%
乙：现期 210，同比增长 5%

比较基期：

甲基期=2201.1=200甲基期=\frac{220}{1.1}=200 甲基期=1.1220​=200

乙基期=2101.05=200乙基期=\frac{210}{1.05}=200 乙基期=1.05210​=200

这里刚好相同。
若乙现期再大一点，比如 212，则乙基期会更大。
所以做题时可直接用“现期大且增速小，基期大”快速判断。

# 5. 基期和差

甲今年 300，同比增长 20%；乙今年 200，同比增长 10%。求去年两者之和。

先看今年和：

300+200=500300+200=500 300+200=500

300÷1.2=250300\div1.2=250 300÷1.2=250

200÷1.1≈182200\div1.1\approx182 200÷1.1≈182

250+182=432250+182=432 250+182=432

# 易错提醒

基期量的分母一定是 1+r1+r1+r，不是只除以 rrr。
“求过去的量”，优先往基期量公式上靠。

# 2. 现期量（考得较少）

# 识别

  • 求未来的量
  • 求今年、当期的量

# 公式

  1. 现期量=基期量+增长量现期量 = 基期量 + 增长量现期量=基期量+增长量
  2. 现期量=基期量×(1+r)现期量 = 基期量 \times (1+r)现期量=基期量×(1+r)

# 技巧

  • 注意问法：是保持增量，还是保持增速
  • 若材料未知增长率，可结合倍数关系和选项快速估算
  • 当现期和基期很接近（1 倍出头）时，可借助增长量估算，通常选略大的结果

# 方法示例

# 1. 直接乘 (1+r)(1+r)(1+r)

某地区去年游客接待量为 250 万人，今年同比增长 12%，求今年游客接待量。

现期量=250×(1+12%)=250×1.12=280现期量=250\times(1+12\%)=250\times1.12=280 现期量=250×(1+12%)=250×1.12=280

# 2. 保持增量 vs 保持增速

某企业去年产值 100 万元，今年 120 万元。

则今年比去年增长量为：

120−100=20120-100=20 120−100=20

增长率为：

20100=20%\frac{20}{100}=20\% 10020​=20%

若明年保持增量不变：

120+20=140120+20=140 120+20=140

若明年保持增速不变：

120×1.2=144120\times1.2=144 120×1.2=144

两种问法答案不同。

# 3. 增长率未知，用倍数和选项估算

某厂去年产量 80 吨，今年比去年“增加到原来的 1.5 倍”，则现期量直接为：

80×1.5=12080\times1.5=120 80×1.5=120

这种题不必先转成增长率 50%，直接看倍数更快。

# 4. 现期与基期很接近，用增长量估算

去年 1000，今年增长率仅 2%，求今年值。

现期量=1000×1.02=1020现期量=1000\times1.02=1020 现期量=1000×1.02=1020

“未来的量”不一定都要乘 (1+r)(1+r)(1+r)，有时题目给的是“保持增量不变”。
一定先分清是“增速不变”还是“增量不变”。

3961210×121108\frac{396}{1210}\times\frac{121}{108} 1210396​×108121​

再做等比例简化，减少硬算量。

# 易错提醒

基期比重不是简单用现期比重代替，必须乘上修正项。
看到“去年占比”，一定比现期比重多一步修正。

# 3. 两期比重（必考）

# 识别

  • 问今年比重与去年相比
  • 问比重上升还是下降、上升下降多少

# 公式

  • AB×a−b1+a\frac{A}{B} \times \frac{a-b}{1+a}BA​×1+aa−b​

# 技巧

  1. 判断升降：若 a>ba>ba>b，则上升；若 a<ba<ba<b，则下降
  2. 定大小：比重差 <∣a−b∣< |a-b|<∣a−b∣
  3. 若范围仍无法选出，再代公式计算

# 方法示例

# 1. 判断升降

部分增速 15%，总体增速 10%，因为：

15%>10%15\%>10\% 15%>10%

所以今年比重 上升。

若部分增速 8%，总体增速 12%，则比重 下降。

# 2. 比重差小于 ∣a−b∣|a-b|∣a−b∣

若 a=15%a=15\%a=15%，b=10%b=10\%b=10%，则：

∣a−b∣=5%|a-b|=5\% ∣a−b∣=5%

比重变化量一定 小于 5 个百分点。

# 3. 代公式

现期比重为 40%，部分增速 15%，总体增速 10%。

比重变化=40%×15%−10%1+15%比重变化=40\%\times\frac{15\%-10\%}{1+15\%} 比重变化=40%×1+15%15%−10%​

=40%×5%1.15≈1.74%=40\%\times\frac{5\%}{1.15}\approx1.74\% =40%×1.155%​≈1.74%

即比重大约 上升 1.7 个百分点。

# 易错提醒

比重变化不是直接用 a−ba-ba−b，而是和现期比重有关。
先判断升降，再决定要不要细算。

# 五、平均数问题

# 1. 现期平均（必考）

# 识别

  • 求今年的平均数
  • 求单位量、每人、每户、每家、每吨等

# 公式

  • 平均数=后(A)前(B)平均数=\frac{后(A)}{前(B)}平均数=前(B)后(A)​

# 技巧

  1. 截位直除
  2. 多数加和求平均时，可用削峰填谷

# 方法示例

# 1. 现期平均

某地今年旅游总收入为 96 亿元，接待游客 1200 万人次。

平均数=96亿元1200万人次=800元/人次平均数=\frac{96亿元}{1200万人次}=800元/人次 平均数=1200万人次96亿元​=800元/人次

# 2. 削峰填谷

四个数为：9、10、11、14，求平均数。
可把 14 多出的 3 分给 9、10、11：

9,10,11,14→11,11,11,119,10,11,14 \rightarrow 11,11,11,11 9,10,11,14→11,11,11,11

平均数即为 11。

# 易错提醒

平均数问题要先找准“总量”和“份数”。
谁在前、谁在后，先看单位。

# 2. 基期平均（很少考）

# 识别

  • 求去年的平均数

# 公式

  • AB×1+b1+a\frac{A}{B} \times \frac{1+b}{1+a}BA​×1+a1+b​

# 技巧

  • 差距大，上下截两位约分
  • 差距小，可按基期比重的思路处理

# 方法示例

# 1. 直接代公式

今年总产值 480 万元，同比增长 20%；职工人数 110 人，同比增长 10%。求去年人均产值。

基期平均=480110×1+10%1+20%基期平均=\frac{480}{110}\times\frac{1+10\%}{1+20\%} 基期平均=110480​×1+20%1+10%​

≈4.364×1.11.2≈4\approx4.364\times\frac{1.1}{1.2}\approx4 ≈4.364×1.21.1​≈4

去年人均产值约为 4 万元/人。

# 2. 差距大，上下截两位

若算：

684228×1.051.20\frac{684}{228}\times\frac{1.05}{1.20} 228684​×1.201.05​

可先把：

684228≈6823\frac{684}{228}\approx\frac{68}{23} 228684​≈2368​

再配合后半部分一起估算，速度更快。

# 易错提醒

基期平均数与基期比重形式相同，但含义不同。
公式相同，关键是先识别题型。

# 3. 两期平均（必考）

# 识别

  • 问今年平均数与去年相比
  • 求平均数增长量、增长率

# 公式

  • 平均数的增长量=AB×a−b1+a平均数的增长量=\frac{A}{B} \times \frac{a-b}{1+a}平均数的增长量=BA​×1+aa−b​
  • 平均数的增长率=a−b1+b平均数的增长率=\frac{a-b}{1+b}平均数的增长率=1+ba−b​

# 技巧

  1. 增长量考得少，时间紧可先判断升降，再猜一个较小值
  2. 增长率直接代公式，通常属于送分题

# 方法示例

# 1. 平均数增长率

总收入增长率为 18%，人数增长率为 8%，求人均收入增长率。

平均数增长率=18%−8%1+8%=10%1.08≈9.26%平均数增长率=\frac{18\%-8\%}{1+8\%} =\frac{10\%}{1.08}\approx9.26\% 平均数增长率=1+8%18%−8%​=1.0810%​≈9.26%

# 2. 增长量先判断升降

若总量增速 12%，份数增速 5%，则平均数一定上升。
若选项为 0.8、1.6、3.5、8.9，一般不会特别大，优先考虑较小正值。

# 易错提醒

平均数增长率不是直接用 a−ba-ba−b，还要除以 (1+b)(1+b)(1+b)。
看到“两期平均增长率”，直接套公式最稳。

# 六、倍数问题

# 1. 现期倍数（必考）

# 识别

  • 求今年的倍数关系
  • 求 A 是 B 的多少倍

# 公式

  • 倍数=前(A)后(B)倍数=\frac{前(A)}{后(B)}倍数=后(B)前(A)​

# 注意

  1. AAA 比 BBB 多几倍：=AB−1=\frac{A}{B}-1=BA​−1
  2. AAA 超过 BBB 的 nnn 倍：即 A>n×BA > n \times BA>n×B
  3. 若总量相同，可用比重代替倍数计算

# 方法示例

# 1. 是多少倍

甲产品产量 360 吨，乙产品 120 吨。

倍数=360120=3倍数=\frac{360}{120}=3 倍数=120360​=3

甲是乙的 3 倍。

# 2. 多几倍

仍以上题为例：

360120−1=2\frac{360}{120}-1=2 120360​−1=2

所以甲比乙 多 2 倍。

# 3. 超过 B 的 n 倍

若问甲是否超过乙的 2 倍：

360>2×120360 > 2\times120 360>2×120

成立，所以“超过乙的 2 倍”。

# 4. 总量相同时用比重

某企业总收入相同，甲产品占 40%，乙产品占 20%，则甲是乙的：

40%20%=2\frac{40\%}{20\%}=2 20%40%​=2

即 2 倍。

# 易错提醒

“是几倍”与“多几倍”不是一回事，后者要再减 1。
做倍数题时，先看题干到底问哪一种。

# 2. 基期倍数（很少考）

# 识别

  • 求去年的倍数关系

# 公式

  • AB×1+b1+a\frac{A}{B} \times \frac{1+b}{1+a}BA​×1+a1+b​

# 注意

  • 处理方法与基期平均数基本一致

# 方法示例

今年甲行业增加值为 300 亿元，同比增长 20%；乙行业增加值为 150 亿元，同比增长 50%。求去年甲是乙的多少倍。

基期倍数=300150×1+50%1+20%=2×1.51.2=2.5基期倍数=\frac{300}{150}\times\frac{1+50\%}{1+20\%} =2\times\frac{1.5}{1.2}=2.5 基期倍数=150300​×1+20%1+50%​=2×1.21.5​=2.5

所以去年甲是乙的 2.5 倍。

# 易错提醒

基期倍数和现期倍数不要混用。
只要问“去年是几倍”，就要补修正项。

# 七、间隔增长率

# 方法概述

间隔增长率常见于“隔一年”或“跨两年”的题目，本质是连续两期增长率的合成。

# 识别

  • 时间上隔一年
  • 求增长率 / 倍数 / 基期

# 公式

  • r间=r1+r2+r1r2r_{间}=r_1+r_2+r_1r_2r间​=r1​+r2​+r1​r2​
  • 即：和 + 积

# 技巧

  1. 计算通常不难，结合选项即可
  2. 间隔倍数=1+间隔r间隔倍数 = 1 + 间隔r间隔倍数=1+间隔r
  3. 间隔基期=现期1+间隔r间隔基期 = \frac{现期}{1+间隔r}间隔基期=1+间隔r现期​
  4. 逆向考查：r2=r间−r11+r1r_2=\frac{r_{间}-r_1}{1+r_1}r2​=1+r1​r间​−r1​​

# 方法示例

# 1. 间隔增长率

2024 年同比增长 10%，2025 年同比增长 20%。

r间=10%+20%+10%×20%=32%r_{间}=10\%+20\%+10\%\times20\% =32\% r间​=10%+20%+10%×20%=32%

# 2. 间隔倍数

由上题可知：

间隔倍数=1+32%=1.32间隔倍数=1+32\%=1.32 间隔倍数=1+32%=1.32

说明 2025 年是 2023 年的 1.32 倍。

# 3. 间隔基期

若 2025 年现期量为 264，间隔增长率为 32%，则 2023 年量为：

2641.32=200\frac{264}{1.32}=200 1.32264​=200

# 4. 逆向考查

若间隔增长率为 32%，已知第一年增长率为 10%，求第二年增长率：

r2=32%−10%1+10%=22%1.1=20%r_2=\frac{32\%-10\%}{1+10\%} =\frac{22\%}{1.1}=20\% r2​=1+10%32%−10%​=1.122%​=20%

# 易错提醒

间隔增长率不是简单相加，还要加上乘积项。
看到“隔年增长率”，第一反应就是“和 + 积”。

# 八、年均增长率

# 方法概述

年均增长率强调的是：在若干年中平均每年增长多少。

# 识别

  • 出现“年均”“平均每年增长率”

# 公式

  • (1+r)n=现期量基期量(1+r)^n = \frac{现期量}{基期量}(1+r)n=基期量现期量​

# 技巧

  1. 比较时，重点看 现期量基期量\frac{现期量}{基期量}基期量现期量​
  2. 计算时，可用居中代入快速试值

# 方法示例

# 1. 直接计算

某产业 2021 年产值为 100 亿元，2025 年产值为 146.41 亿元，求年均增长率。

现期量基期量=1.4641\frac{现期量}{基期量}=1.4641 基期量现期量​=1.4641

又因为经历 4 年：

(1+r)4=1.4641(1+r)^4=1.4641 (1+r)4=1.4641

观察得：

1.14=1.46411.1^4=1.4641 1.14=1.4641

所以：

r=10%r=10\% r=10%

# 2. 比较时看倍数

甲：4 年从 100 增到 146
乙：4 年从 100 增到 160

因为：

160100>146100\frac{160}{100}>\frac{146}{100} 100160​>100146​

所以乙的年均增长率更高。

# 3. 居中代入

若 3 年后总倍数约为 1.33，可先试：

1.13=1.3311.1^3=1.331 1.13=1.331

说明年均增长率约为 10%。

# 易错提醒

年均增长率不是把总增长率直接除以年数。
看到“年均”，就不要做简单平均。

# 九、混合增长率

# 方法概述

混合增长率是资料分析中的高频重点题型，常见于“整体由几个部分构成”的情况。

# 识别

  • 有加和关系
  • 求其中某一部分或总体的增长率

# 技巧

  1. 混合总体居中：最小r<总体r<最大r最小r < 总体r < 最大r最小r<总体r<最大r
  2. 偏向量较大的：总体增速离基数大的更近
  3. 线段法拓展：距离与量成反比

# 方法示例

# 1. 居中原则

甲增长率 10%，乙增长率 30%，则总体增长率一定满足：

10%<总体增长率<30%10\%<总体增长率<30\% 10%<总体增长率<30%

不可能跑到这两个范围之外。

# 2. 偏向基数大的

甲收入 80 万元，增长率 10%；乙收入 20 万元，增长率 30%。
因为甲基数更大，所以总体增长率会更靠近 10%，而不会靠近 30%。

# 3. 直接算验证

甲增长量：

80×10%=880\times10\%=8 80×10%=8

乙增长量：

20×30%=620\times30\%=6 20×30%=6

总基期量：

80+20=10080+20=100 80+20=100

总增长量：

8+6=148+6=14 8+6=14

总体增长率：

14100=14%\frac{14}{100}=14\% 10014​=14%

果然介于 10% 与 30% 之间，且更接近 10%。

# 4. 线段法理解“距离与量成反比”

若总体增长率在 14%，则它到 10% 的距离是 4，到 30% 的距离是 16。
距离比为：

4:16=1:44:16=1:4 4:16=1:4

反过来说明对应基数约为：

4:14:1 4:1

即低增速那一侧的基数更大。

# 易错提醒

混合增长率绝不会跑到最大值和最小值外面去。
先判断“居中”，再看偏向谁，很多题不用细算。

# 十、总结

# 高频优先级

资料分析里最常考、最该优先掌握的是：

  • 截位直除
  • 基期量
  • 增长量
  • 增长率
  • 现期比重
  • 两期比重
  • 现期平均
  • 现期倍数
  • 混合增长率

# 复习建议

  1. 先熟公式，再练识别
  2. 每个题型至少掌握一种速算思路
  3. 做题时先看选项差距，再决定是精算还是估算

# 最后提醒

资料分析不是“算得多快”，而是“识别得够不够准、方法选得对不对”。
会判断、会估算、会排除，往往比埋头硬算更重要。
